# Deployer
Deploy multiple sloth applications in a VM using docker.

## Installation
> git clone <url> /opt/deployer

## Initialization
> cd /opt/deployer
> 
> ./start.sh

## Keys
> uuidgen
> 
> vim keys.yaml
> 
> XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX: https://github.com/XXXXX/XXXXX.git

## Stop
> cd /opt/deployer
> 
> ./stop.sh
